Для запуска вашего проекта вам не нужен файл requiments.txt (вероятно, вы имели в виду requirements.txt). Стандартный файл называется                
requirements.txt.                                                                                                                                    

Вот список зависимостей, которые нужно добавить в этот файл, основанный на импортах в вашем коде:                                                    

                                                                                                                                                     
 openai                                                                                                                                              
 rich                                                                                                                                                
 argparse                                                                                                                                            
 readline                                                                                                                                            
 locale                                                                                                                                              
 os                                                                                                                                                  
 sys                                                                                                                                                 
                                                                                                                                                     

Как использовать:                                                                                                                                    

 1 Создайте файл requirements.txt в той же папке, где лежит main.py.                                                                                 
 2 Скопируйте туда строки выше.                                                                                                                      
 3 Установите зависимости командой:                                                                                                                  

                                                                                                                                                     
 pip install -r requirements.txt                                                                                                                     
                                                                                                                                                     

Примечание: В коде используются модули стандартной библиотеки Python (os, sys, argparse, locale, readline), они не требуют установки через pip.      Для запуска вашего проекта вам не нужен файл requiments.txt (вероятно, вы имели в виду requirements.txt). Стандартный файл называется
