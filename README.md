# llama-cli-client

Легковесный, минималистичный и отзывчивый терминальный клиент для работы с локальными большими языковыми моделями (LLM). Он полностью совместим со спецификацией OpenAI API и оптимизирован под работу в связке с локальными движками, такими как `llama.cpp` (llama-server).

> **💡 Дисклеймер:** Этот проект создавался исключительно в учебных целях для прокачки навыков программирования на Python (разбор сетевого взаимодействия, обработка системных прерываний, работа с кодировками ввода-вывода и парсинг аргументов CLI). Тем не менее, утилита получилась полностью стабильной, быстрой и ежедневно используется автором для решения повседневных задач и автоматизации.

---

## 🚀 Основные возможности (Features)

- **Интерактивный Markdown:** Динамический стриминг токенов в реальном времени красиво рендерится и подсвечивается прямо в терминале благодаря библиотеке `rich`.
- **Два режима работы:** 
  - Полноценный интерактивный чат с удержанием истории контекста.
  - Одноразовый запрос (Single-shot) через аргументы командной строки без запуска бесконечного цикла.
- **Умная интеграция файлов (`-f` / `--file`):** Возможность передать файл вместе с текстовым промптом одной командой — идеально подходит для проведения код-ревью или быстрого анализа логов.
- **Отказоустойчивость:** Жесткий перехват прерываний (`Ctrl+C`) плавно прерывает генерацию без вылета программы, а принудительное форматирование кодировки потоков полностью защищает инпут от падений на кириллице.

## 🛠 Требования

- **Python 3.10** или выше
- Запущенный локальный или удаленный сервер с OpenAI-совместимым API (например, `llama.cpp` на дефолтном порту 8080)
