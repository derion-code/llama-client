# llama-cli-client

Легковесный терминальный клиент для работы с локальными и удалёнными LLM через **OpenAI Compatible API**.

Проект ориентирован на максимальную простоту, минимальное количество зависимостей и быстрый отклик терминала. Клиент поддерживает потоковую генерацию, Markdown-рендеринг, историю диалога и передачу файлов в запрос.

> Проект создавался как учебный, однако ежедневно используется автором для работы с локальными моделями через `llama.cpp`.

---

## Возможности

- OpenAI Compatible API
- Потоковая генерация (Streaming)
- Красивый Markdown вывод через Rich
- Интерактивный режим общения
- Одноразовые запросы из командной строки
- Передача содержимого файлов модели
- История сообщений внутри сессии
- Безопасное прерывание генерации (`Ctrl+C`)
- Простая архитектура без лишних зависимостей

## Установка

```bash
git clone https://github.com/derion-code/llama-cli-client.git

cd llama-cli-client

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt
```

## Настройка

Перед первым запуском необходимо изменить параметры подключения в `main.py`.

```python
api_url = "http://192.168.1.124:8080/v1"
api_key = "None"

system_prompt = """
Ты краткий и полезный ассистент.
"""
```

Можно изменить:

- адрес сервера
- API Key
- системный промпт

---

## Запуск сервера

Например, с использованием **llama.cpp**:

```bash
./llama-server \
    -m models/model.gguf \
    --host 0.0.0.0 \
    --port 8080
```

После этого клиент сможет подключиться к серверу.

---

## Запуск клиента

Интерактивный режим

```bash
python main.py
```

После запуска появится приглашение

```text
>>>
```

Введите сообщение и нажмите Enter.

---

## Одноразовый запрос

```bash
python main.py -p "Расскажи про Linux"
```

После получения ответа программа завершится.

---

## Анализ файла

Можно прикрепить файл к запросу.

```bash
python main.py \
    -p "Проведи ревью кода" \
    -f main.py
```

Содержимое файла автоматически добавляется в запрос в виде Markdown-блока.

## Команды

Во время интерактивной работы доступны команды.

| Команда | Описание |
|----------|----------|
| `/help` | список команд |
| `/about_program` | информация о программе |
| `/clear` | очистить терминал |
| `/clear_history` | очистить историю сообщений |
| `/stream` | включить/выключить Streaming |
| `/markdown` | включить/выключить Markdown |
| `/exit` | выход |
| `/quit` | выход |

---

## Управление

| Клавиша | Действие |
|----------|----------|
| Enter | отправить сообщение |
| Ctrl+C | остановить генерацию |
| Ctrl+D | завершить программу |
